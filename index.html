關於我

我是 陳昱瑄，畢業於 銘傳大學電子工程學系。個性樂觀、善於團隊合作，曾擔任社團活動長，培養了溝通協調能力。

畢業後，我意識到自己的技能需要提升，因此參加了 PHP 全端開發課程，學習 PHP + MySQL，並取得 網頁設計乙級證照。結訓後，我在 國立清華大學 擔任約聘 PHP 工程師，負責 全國中小學人力資源系統 的開發與維護，並開始接觸 Vue.js 和 Laravel。

後來我加入 全球遊戲公司與恒發數位科技，擔任 PHP 後端工程師，負責： API 串接（第三方遊戲平台、金流整合） 高併發資料庫優化 自動化報表開發 Linux 環境部署與維護 在這段時間內，我強化了 PHP 與 MySQL 的應用，並累積了 高併發處理、系統整合 的實戰經驗。

離開上一份工作後，我意識到前端能力的重要性，因此進修 Vue 3 和 Node.js，完成了 前後端分離專案，並掌握 Express.js、MongoDB、Redis 等技術，除了強化了全端開發的能力，也讓我對於學習新的框架及語言這方面更有自信與把握。

我希望未來能在新的環境中發揮專長，為團隊貢獻價值並且找到屬於自己的歸屬。
